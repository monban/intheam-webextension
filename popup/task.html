inthe.am Extension Set-up

Welcome to the inthe.am browser extension! We need your inthe.am API key in order to send tasks to your inthe.am task list.

  1. Log in or create an account on inthe.am
  2. Open the configuration page by clicking 'Configuration' in the upper right of the main dashboard
  3. Expand the 'API Access' section of the inthe.am configuration page
  4. Select (by double clicking, on most systems) the API key
  5. Click the browser action button () again to save the API key
  6. You will be taken to the add-on configuration page. If you'd like, specify a default project or tags to be added to tasks created from the browser
  7. Click 'Save'
  8. Now you can create tasks from any website by simply selecting a few words, and pressing the extension action button ()
