inthe.am Extension Set-up

Welcome to the inthe.am browser extension! We need your inthe.am API key in order to send tasks to your inthe.am task list.

  1. Log in to inthe.am
  2. Click here to go to the configuration page and follow the instructions there

Be sure to check out the options page for this browser extension, you can tweak default tags, project, or enter the API key manually.

Now you can create tasks from any website by simply selecting a few words, and pressing the extension action button ()
